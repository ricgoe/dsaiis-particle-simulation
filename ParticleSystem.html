move_particles(skip_interaction: bool = False)[source]ÔÉÅ

Moves particles for one timestep and updates their velocities based on collision responses. The velocity is set in the constructor. The energy stored in the system stays the same, because only the angles change.

The method uses the current velocity to update particle positions, detects collisions, and then updates the velocity for each colliding pair by calling update_velocities_collisions. The particles‚Äô positions are then updated with the velocities changed in their angles.
